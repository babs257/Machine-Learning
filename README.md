## Projects

### 1. Customer Review Sentiment Analysis with Time-Series Tracking
**File:** `Customer_Review_Sentiment_NLP_TimeSeries.ipynb`  
NLP pipeline using fine-tuned BERT and BERTopic for sentiment classification and 
topic modelling on real customer review data. Includes time-series analysis tracking 
sentiment trends over time to support root-cause analysis of changing customer 
perceptions.  
**Skills:** Python, NLP, BERT, BERTopic, LDA, Matplotlib, Plotly, Time-Series Analysis

---

### 2. Nando's Review Analysis — MVP Sentiment & Time-Series
**File:** `Nandos_Review_Sentiment_TimeSeries.ipynb`  
End-to-end sentiment analysis and temporal trend modelling on restaurant review data. 
Demonstrates NLP preprocessing, model evaluation, and interactive visualisation of 
results over time.  
**Skills:** Python, NLP, Sentiment Analysis, Time-Series, Plotly, pandas

---

### 3. Applied Machine Learning — Classification & Regression
**File:** `Applied_Machine_Learning.ipynb`  
Comprehensive applied ML notebook covering supervised learning techniques including 
classification, regression, and model evaluation. Includes feature engineering, 
cross-validation, hyperparameter tuning, and performance analysis using scikit-learn 
and XGBoost.  
**Skills:** Python, scikit-learn, XGBoost, Feature Engineering, Model Evaluation

---

### 4. Demand Forecasting — Sticker Sales Time-Series Prediction
**File:** `Demand_Forecasting_TimeSeries.ipynb`  
Time-series forecasting project predicting product demand using statistical and 
machine learning approaches. Covers stationarity testing, decomposition, and 
evaluation of forecast accuracy.  
**Skills:** Python, Time-Series Forecasting, pandas, Matplotlib, Statistical Modelling
